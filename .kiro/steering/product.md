# Product Overview

This is a Streamlit web application that generates personalized activity recommendations for users (Preston and Isaac). The app reads data from Google Sheets to provide:

- Daily task suggestions with durations and descriptions
- Workout routines with exercises, sets, and video links
- Bonus random tasks

The application provides an interactive interface where users select their name and receive a curated list of activities for the day.
